# Άρθρο 361 - Κώδικας Πολιτικής Δικονομίας

Οι διάδικοι οφείλουν να βοηθούν για την ενέργεια της αυτοψίας και να κάνουν ό,τι είναι αναγκαίο για τη διεξαγωγή της.
