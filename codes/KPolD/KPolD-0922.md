# Άρθρο 922 - Κώδικας Πολιτικής Δικονομίας

Όποιος έχει δικαίωμα να ενεργήσει αναγκαστική εκτέλεση μπορεί, όταν πρόκειται να την ενεργήσει, να ζητήσει την παροχή κληρονομητηρίου για το δικαίωμα εκείνου κατά που οποίου θα στραφεί η εκτέλεση.
