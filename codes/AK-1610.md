# Άρθρο 1610 - Αστικός Κώδικας - Πρόσθετες εγγυήσεις για τον ανήλικο

Η απόφαση του δικαστηρίου για την εισαγωγή του ανηλίκου σε ειδικό ίδρυμα ή κατάστημα ισχύει γιο έξι μήνες. Η ισχύς της μπορεί να παρατείνεται για έξι μήνες κάθε φορά. Η απόφαση μπορεί να ανακαλείται οποτεδήποτε, αν εκλείψουν οι λόγοι που επέβαλαν τη λήψη αυτού του μέτρου.
