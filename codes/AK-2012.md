# Άρθρο 2012 - Αστικός Κώδικας - Πρόσωπο για το οποίο γίνεται η παροχή

Ο διαθέτης μπορεί να τάξει τρόπο για ορισμένο σκοπό και να αφήσει στο βεβαρημένο ή σε τρίτον να προσδιορίσει το πρόσωπο στο οποίο θα γίνει η παροχή.
